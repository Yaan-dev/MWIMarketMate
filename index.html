合规性公示

本插件不包含任何自动购买功能——自动操作违反游戏规则第 4.2 条。它只负责帮你找到物品、算清数量,剩下的交给你来完成。

✓ 本插件做的事

在客户端跳转到某个物品的市场页面——与你手动 Shift 点击背包物品效果完全一致。一次点击,一次请求。不信任跳转?也可以只用市场内高亮。

✗ 违规插件做的事

一键完成多种物品的批量购买,向服务器批量发送请求——这是自动化操作,本插件不做,永远不做。

所有购买行为均需玩家手动操作。

FEATURES · 功能

从"缺多少"到"买回来",中间的一切

打开任意制作面板,材料行上直接长出徽章;缺的东西一键进清单,清单帮你逐个跳到市场——每一步都看得见,每一步都是你在操作。

CALC

缺料计算

直读游戏数据层,不解析屏幕文本,精确到个位。工匠茶、暴饮袋自动减免,可选 Z-score 安全边际(90%–99.9%)。

COVERAGE

全面板覆盖

制作、炼金(点金/分解/转化)、房屋建造,以及从市场、库存任意位置打开的详情弹窗——徽章无处不在。

CART

购物清单

桌面侧栏 / 移动抽屉双形态,市场实时价格与预估总价;WebSocket 精确追踪库存,买到自动扣减,购齐自动收起。

MARKET

采购流水线

市场弹窗自动预填数量,购齐横幅提示下一项,自定义快捷键一按直达——全部由你手动触发。

PLANS

计划与升级链

制作计划锁定材料库存,根治多配方共用材料的缺口误算;升级链树形展示,逐步建独立计划,完成自动释放。

DIAG

互操作与诊断

公共 API window.MWIMM 供其他脚本调用;与主流插件同开经专项适配;出问题一句 __log() 导出全部诊断信息。

RELEASE · 2.0.0

2.0.0:架构升级 + 稳定性大修

十余轮候选版打磨后的正式发行版。完整更新日志见 GreasyFork 页面。

全新界面NEW

Shadow DOM 双形态壳,样式与游戏及其他插件彻底隔离;设置即时生效、自动保存。
