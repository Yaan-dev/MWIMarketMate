合规性公示

✓ 已提交管理员审阅

本插件不包含任何自动购买功能——自动操作违反游戏规则第 4.2 条。它只负责帮你找到物品、算清数量,剩下的交给你来完成。

✓ 本插件做的事

在客户端跳转到某个物品的市场页面——与你手动 Shift 点击背包物品效果完全一致。一次点击,一次请求。不信任跳转?也可以只用市场内高亮。

✗ 违规插件做的事

一键完成多种物品的批量购买,向服务器批量发送请求——这是自动化操作,本插件不做,永远不做。

所有购买行为均需玩家手动操作。

FEATURES

从"缺多少"到"买回来",中间的一切

打开任意制作面板,材料行上直接长出徽章;缺的东西一键进清单,清单帮你逐个跳到市场——每一步都看得见,每一步都是你在操作。

CALC ENGINE

直读游戏数据层,不解析屏幕文本。工匠茶、暴饮袋自动减免,可选 Z-score 安全边际(90%–99.9%);三层防注入,与其他插件同开也不出鬼数。

COVERAGE

制作、炼金(点金/分解/转化)、房屋建造,以及从市场、库存任意位置打开的详情弹窗——徽章无处不在。

RELEASE 2.1.0

2.1.0:全面板覆盖,抗插件冲突

在 2.0.0 架构之上的稳定性强化版——补齐炼金 / 强化面板,并根治与第三方插件同开时的徽章冻结。完整更新日志见 GreasyFork 页面。

炼金 / 强化全覆盖FIX

炼金(点金 / 分解 / 转化)与强化面板的徽章和「加入购物车」按钮不再缺失,与制作面板一致覆盖。

抗插件冲突FIX

跳转市场往返后徽章不再冻结或消失,即便与会向面板注入内容的第三方插件同开——根治了刷新被 DOM 风暴「饿死」的根因。

数据层配方识别NEW

改用 React Fiber 读取确定的配方标识,不再依赖屏幕文本或类名推断——识别更稳,与其他插件共存更可靠。

看门狗增强

市场弹窗开 / 关即时刷新、面板复用换配方强制重绘;__log() 新增去抖诊断计数,排查跨插件冲突更快。

INSTALL

两分钟装好

01

安装脚本管理器

浏览器装上 Tampermonkey(篡改猴)扩展,Chrome / Edge / Firefox 均可。

02

安装市场伴侣

03

打开制作面板
